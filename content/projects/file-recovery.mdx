### File Recovery Forensics Tool

A system forensics application designed to parse and reconstruct raw disk image structures, allowing recovery of deleted files from modern Linux filesystems.

#### Key Features:
- **XFS & BTRFS Parsing**: Direct parsing of file system structures without mounting.
- **Inode Reconstruction**: Extracts block pointers from unlinked inodes to trace deleted files.
- **Metadata Recovery**: Recovers directory paths, creation dates, and original filenames.
