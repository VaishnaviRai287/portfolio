### ML Visualizer: Full-Stack Machine Learning Experimentation Platform

ML Visualizer is an interactive web platform designed to let developers configure, train, and evaluate deep learning models in real-time, helping users build deep intuition for AI algorithm behaviors.

#### Core Capabilities:
- **Interactive Training Engine**: Configure, train, and evaluate deep learning models directly through a web browser with real-time UI updates.
- **WebSocket Telemetry Stream**: Streams live metrics, training logs, loss curves, and epoch-by-epoch model parameter updates using Django Channels and WebSockets.
- **Visual Diagnostics**: Dynamically displays classification decision boundaries in geometric space, confusion matrices, ROC-AUC, and Precision-Recall diagnostics.
- **Experiment Comparison**: Side-by-side dashboards to run hyperparameter tuning experiments, compare loss optimization curves, and track training benchmarks.
